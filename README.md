# CSS Dojo 🥋

Practicing CSS from scratch, chapter by chapter using [web.dev/learn/css](https://web.dev/learn/css).

Each folder is a mini project built while learning that topic — no frameworks, no shortcuts.

## Goal

Work through every chapter, build something small, push it. Simple.
